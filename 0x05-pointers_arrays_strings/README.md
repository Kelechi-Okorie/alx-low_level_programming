## Project directory for Pinters_Arrays_Strings
File 0-reset_to_98.c - A program that updates the value pointed by a pionter
File 1-swap.c - A program that swaps the value of 2 integers
File 2-strlen.c - A program that returns the length of a string
File 3-puts.c - A program that prints a string to stdout
File 4-print_rev.c - A program that prints a string, in reverse
File 5-rev_string.c - A program that reserves a string
File 6-puts3.c - A program that prints every other character of a string
