## README file for 0x07 Pointers, Arrays and Strings

File 0-memset.c - A program that fills memory with a constant byte
File 1-memcpy.c - A program that copies memory area
File 2-strchar.c - A program that locates a character in a string
File 3-strspn.c - A program to get the length of a prefix substring
File 4-strpbrk.c - A program that searches a string for any of a set of bytes
File 5-strstr.c - A program that locates a substring
File 7-print_chessboard - A program that prints the chessboard
File 8-print_diagsums.c - A program that prints the sum of the two diagonals of
square matrix of integers
