# Readme file for 0x14-bit_manipulation
File 0-binary_to_uint.c - A program that converts binary number to an unsigned int
file 1-print_binary.c - A program that prints the binary representation of a number
File 2-get_bit.c - A program that returns the value of a bit at a given index
File 3-set_bit.c - A program that sets the value of a bit to 1 at a given index
File 4-clear_bit.c - A program that sets the value of a bit to 0 at a given index
File 5-flip_bits.c - A program that returns the number of bits you would need to 
flip to get from one number to another
File 100-get_endianness.c - A program that checks the endianess of a computer
